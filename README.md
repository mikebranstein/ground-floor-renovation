# Ground Floor Renovation — Design Vision

A comprehensive design presentation website for a ground floor home renovation project in Auckland, New Zealand. This site showcases design recommendations, rendered visions, and detailed specifications for transforming the ground floor into a modern, cohesive living suite.

## Overview

The ground floor is a self-contained suite with enormous potential. Currently held back by decades of dated finishes, dampness, and disconnection from the upper floor's contemporary aesthetic — this project presents a complete transformation vision: coherent, modern, and liveable.

## Features

### Sections Covered
1. **Overview** — Project summary and design principles
2. **Exterior & Entry** — Black aluminium joinery and warm brick entryway
3. **Laundry** — Modern, functional laundry space
4. **Bedroom & Zones** — Sleeping zone with oak flooring and NZ art
5. **Hallway** — Cohesive passage connecting all spaces
6. **Bathroom** — Marble-look tile with brushed gold fittings
7. **Storage Room** — Optimized storage solutions

### Design Principles
- **Palette**: Black, white & warm oak throughout
- **Priority Zero**: Moisture remediation before materials
- **Asbestos First**: Licensed assessment required before floor work

## Technology

Built with vanilla HTML5 and CSS3:
- No JavaScript frameworks or dependencies
- Responsive design with mobile-first approach
- Google Fonts (Cormorant Garamond & DM Sans)
- Smooth scrolling navigation and tab switching
- Optimized for modern browsers

## Usage

Simply open `index.html` in any modern web browser. The site is fully self-contained and requires no build process or server.

```bash
# Open directly in browser
start index.html  # Windows
open index.html   # macOS
```

## Design Details

All design renders showcase a cohesive aesthetic that:
- Reinforces the upper floor's contemporary language
- Uses warm oak-tone LVP flooring throughout
- Features matte black aluminium joinery
- Prioritizes structural remediation (waterproofing, insulation)
- Eliminates heritage detailing and brass fixtures
